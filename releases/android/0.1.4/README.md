# Triple 2 Eye Protection Android 0.1.4

发布时间：2026-06-07

## 安装包

- `Triple2EyeProtection-Android-0.1.4-release.apk`

## 版本信息

- 包名：`com.triple2.eyeprotection`
- `versionCode`：`5`
- `versionName`：`0.1.4`
- 签名：当前测试版使用 debug 签名

## 更新内容

- 新增悬浮窗/显示在其他应用上层权限。
- 到点休息时，如果系统允许悬浮窗，服务会直接显示一个全屏覆盖层。
- 覆盖层可以盖在其他应用上方，包含“休息时间到了”、倒计时和“休息好了”按钮。
- 保留高优先级通知和 full-screen intent 作为兜底。
- 可直接覆盖安装 `0.1.3`。

## 必须开启的权限

安装后打开应用，进入“打开保活设置”，至少确认：

- 允许通知
- 允许悬浮窗/显示在其他应用上层
- 允许全屏提醒
- 允许精确闹钟
- 允许后台活动/自启动

如果未开启悬浮窗权限，华为系统仍可能只显示通知状态变化，而不允许应用盖到 B 站等其他应用上方。

## 测试建议

把用眼时长设为 `1` 分钟，点击“开启护眼”，切到 B 站或其他应用等待到点。预期结果是出现全屏休息覆盖层。
