# Triple 2 Eye Protection Android 0.1.2

发布时间：2026-06-05

## 安装包

- `Triple2EyeProtection-Android-0.1.2-release.apk`

## 版本信息

- 包名：`com.triple2.eyeprotection`
- `versionCode`：`3`
- `versionName`：`0.1.2`
- 签名：当前测试版使用 debug 签名

## 更新内容

- 保留按钮点击视觉反馈：按下变暗、轻微缩小、松手回弹。
- 移除按钮点击触感震动。
- 可直接覆盖安装上一版同签名 APK。

## 注意

如果手机提示签名冲突，说明已安装的同包名应用不是由当前工程同一签名生成，需要先卸载旧版再安装。
