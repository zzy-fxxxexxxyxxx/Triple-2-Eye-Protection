# Triple 2 Eye Protection Android 0.1.3

发布时间：2026-06-05

## 安装包

- `Triple2EyeProtection-Android-0.1.3-release.apk`

## 版本信息

- 包名：`com.triple2.eyeprotection`
- `versionCode`：`4`
- `versionName`：`0.1.3`
- 签名：当前测试版使用 debug 签名

## 更新内容

- 修复应用在后台时，到点休息只震动、不弹出“休息时间到了”页面的问题。
- 拆分通知逻辑：
  - 常驻通知只负责显示后台倒计时。
  - 休息提醒使用独立高优先级通知和 full-screen intent。
- 新增“打开全屏提醒权限”设置入口。
- 可直接覆盖安装 `0.1.2`。

## 测试建议

安装后打开应用，进入“打开保活设置”，确认通知、全屏提醒、精确闹钟和后台活动权限尽量都已允许。

然后把用眼时长设为 `1` 分钟，点击“开启护眼”，切到 B 站或其他应用，等待到点。预期结果是手机震动并弹出“休息时间到了”页面。
