# Triple 2 Eye Protection Android 0.1.8

Release date: 2026-06-16

## Artifact

- `Triple2EyeProtection-Android-0.1.8-release.apk`

## Changes

- Added a Quick Settings tile named `Triple 2 护眼`.
- Tapping the tile starts or stops the eye protection foreground service.
- When the service is running, the existing foreground notification keeps a status-bar icon visible for quick liveness checks.
- Bumped Android version to `versionCode 9` / `versionName 0.1.8`.

## Notes

- After installation, edit the Android/Huawei quick settings panel and add the `Triple 2 护眼` tile manually.
- This APK keeps the same application id and debug signing setup as previous test builds, so it can upgrade compatible earlier test versions directly.
